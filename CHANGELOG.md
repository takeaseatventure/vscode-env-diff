# Changelog

All notable changes to the "Env Diff" extension will be documented in this file.

## [0.1.0] — 2026-06-25

### Added
- Compare two `.env` files with structured diff output
- Check `.env` against `.env.example` for missing variables
- Sort `.env` file alphabetically in-place
- Support for all standard `.env` formats (quotes, export, inline comments, multiline)
- Zero runtime dependencies
